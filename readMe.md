### For clonning the repository: 
1. git clone https://github.com/Lubka93/cypressE2E_TrackMyCals.git
2. Navigate to cd cypressE2E_TrackMyCals
3. run npm install ()
4. run npm run runner (cypress open) or npm run runnerAll (cypress run)
